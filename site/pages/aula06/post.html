O que é:


O Angular Material é uma implementação do Material Design no Angular feita pela Google. Ele é composto por diversos componentes, bibliotecas CSS e outros elementos, e, faz parte do gerenciador de pacotes do Node o NPM, podendo ser considerado mais um recurso que você pode adicionar ao seu projeto Angular. Link da documentação pode ser acessado aqui

Instalação

Para efetuar a instalação do Angular Material, basta executar o comando "ng add angular/material" Irá abrir uma tela de predefinições da aplicação.

Selecione a segunda opção “Deep Purple/Amber” nesta etapa, é apenas a definição das cores que sua aplicação angular irá apresentar.

Aceite a opção de deixar global a tipografia da aplicação.

E por fim aceite que seja incluído as animações do Angular Material. Agora é só aguardar encerrar a instalação (pode demorar um pouco).

Para criar os botões no html basta selecionar a página que deseja inseri-los e escrever o código do botão aonde você quiser colocar. Após inserir o código no html, pegar o código da aba CSS e inserir na tag <styles> ou no arquivo css da sua aplicação.
