O que é:


É um framework de código criado para o desenvolvimento de aplicativos web baseados em uma página dinâmica, que se reescreve com novos dados emitidos do servidor. Qualquer aplicação angular necessita de um modulo e um componente no mínimo, e a maioria das criações são componentes que se comportam de jeitos diferentes na aplicação. Nesta aula veremos:
-Controller recebendo objetos de ciclos de pagamento.
-Método que busca todos os ciclos.
-Busca pelo ID.

TYPESCRIPT

Surgiu como um superste do JavaScript, com o objetivo de adicionar a este novas funções que não possui nativamente, ou requererem grande esforço para utilização, incluindo uma forte tipagem e a própria orientação a objetos.
Uma das grandes vantagens da TypeScript é permitir que os IDEs proporcionem um ambiente mais rico para detectar erros comuns enquanto você digita o código usando o recurso Intellisense, além de apresentar recursos como os tipos, objetos e funções.
É a linguagem oficial utilizada pelo Angular, comportando toda programação em si realizada neste dentro de seus componentes.

INJEÇÃO DE DEPENDÊNCIA

É um é um padrão de projeto usado para evitar o alto nível de acoplamento de código dentro de uma aplicação ou seja, basicamente é um padrão, ajuda a eliminar o forte acoplamento entre os módulos da aplicação.
Tendo um menor acoplamento de código ajuda o sistema com o aumento da facilidade de manutenção ou inserção de novas funções principalmente.
Como funciona: quando falamos em dependência, estamos falando da necessidade de um arquivo ou objeto externo que será preciso para a aplicação.

HTTPCLIENT

O HttpClient é usado para se comunicar entre o cliente e servidor usando o protocolo HTTP. Em outras palavras, se você deseja usar dados de uma API externa, o HttpClient facilitará essa comunicação por meio de vários métodos disponíveis:
● post()
● get()
● put()
● delete()
● patch()
● request()
● head()
● jsonp()
● options()
HttpClient usa a interface XMLHttpRequest, que oferece suporte a navegadores mais antigos, além de ser fácil de usar.
